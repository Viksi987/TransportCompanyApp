1. Общо описание

TransportCompanyApp е конзолно Java приложение за управление на транспортна компания.
Приложението използва MySQL база данни и реализира основните функционалности, необходими за управление на клиенти, служители, превозни средства и превози.

Архитектурата на проекта е разделена на слоеве:

Entity

DAO

Service

Console (UI)

2. Използвани технологии

Java 17

MySQL 8

JDBC

Maven

IntelliJ IDEA

Git & GitHub

3. Структура на проекта
src/main/java/org/informatica
│
├── app        → конзолно меню и потребителски вход
├── config     → JDBC конфигурация
├── dao        → DAO интерфейси и реализации
├── entity     → бизнес обекти (Client, Employee, Transport, Vehicle, Company)
├── service    → бизнес логика

7. Работа с база данни

Данните се съхраняват в MySQL база данни

В папка database/ е включен SQL dump файл:

database/transport_company.sql


който позволява лесно възстановяване на базата.

8. Стартиране на приложението

Импортирайте проекта в IntelliJ IDEA

Създайте база данни чрез transport_company.sql

Настройте връзката в DBConnection.java

Стартирайте класа:

Main.java

9. Заключение

TransportCompanyApp представлява цялостно конзолно приложение, което покрива изискванията за работа с база данни, обектно-ориентирано програмиране, сортиране, филтриране и генериране на справки.

Проектът е подходящ за учебни цели и демонстрира добри практики при разработка на Java приложения с MySQL.
