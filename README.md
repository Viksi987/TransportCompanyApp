# TransportCompanyApp

## Описание
TransportCompanyApp е конзолно Java приложение за управление на транспортна компания.
Приложението позволява работа с клиенти, служители, превозни средства, превози и компании,
като използва MySQL база данни и JDBC.

## Основни функционалности
- Добавяне, изтриване и показване на клиенти
- Управление на служители (шофьори, мениджъри)
- Управление на превозни средства
- Създаване и управление на превози
- Отбелязване на превоз като платен
- Сортиране и филтриране на данни
- Генериране на справки и приходи
- Запис на превозите във файл

## Технологии
- Java
- MySQL
- JDBC
- Maven
- IntelliJ IDEA

## База данни
SQL файлът за създаване на базата се намира в папка:
database/transport_company.sql


## Стартиране
1. Импортирайте базата данни от SQL файла
2. Настройте връзката в `DBConnection.java`
3. Стартирайте `Main.java`
